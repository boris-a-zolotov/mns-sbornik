\task{Делить и резать, резать и делить}
\begin{itemize}

\itA Нужно провести две диагонали — четыре треугольника, на которые они поделят прямоугольник, будут иметь равную площадь.

\itB Смотреть рисунок:

\def\sq#1{
	\filldraw[fill=gray,draw=black]
		#1 rectangle ++(0.5,0.5);}

\begin{center} \tikz{
	\foreach \x in {0,0.5,1} {
		\sq{(\x,0)} \sq{(\x,1/2)} \sq{(\x,3/2)} \sq{(\x,4/2)}};
	\foreach \y in {0,1,0.5,1.5} {
		\sq{(-1,\y)}};
	\foreach \y in {0,1.5} \sq{(-0.5,\y)};
	\draw[very thick, dashed] (0,3.25) -- (0,-0.75);
} \end{center}

\itC Проведём в круге два диаметра под углом $45^\circ$. Каждый из них поделит круг на две равных части, однако вдвоём они делят круг на 4 части, площади которых равны $\tfrac{1}{8}$, $\tfrac{3}{8}$, $\tfrac{1}{8}$, $\tfrac{3}{8}$ площади круга.
\end{itemize}
